# Cherry Studio Tutorial: Multi-Provider AI Desktop Workspace for Teams

> Learn how to use `CherryHQ/cherry-studio` to run multi-provider AI workflows, manage assistants, and integrate MCP tools in a desktop-first productivity environment.

## Why This Track Matters

Cherry Studio is a high-traction desktop AI workspace that combines many provider options, assistant workflows, data tooling, and MCP support in one cross-platform app.

This track focuses on:

- configuring providers and assistant workflows
- leveraging document and tool integrations
- operating Cherry Studio in team and enterprise contexts
- contributing safely to the rapidly evolving codebase
