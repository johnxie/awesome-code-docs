# Chapter 4: Integration Platforms

Skills can run across Claude Code, Claude.ai, and API-backed systems.

## Claude Code Integration

- Store skills in configured skills directories.
- Call skills directly in coding workflows.
- Combine with local scripts and repository context.

## Claude.ai Integration

- Use skills for repeatable writing and analysis tasks.
- Keep output constraints embedded in instructions.
- Version your skills in Git to track changes.

## API Integration Pattern

- Inject skill instructions into system/developer context.
- Provide structured task input payloads.
- Enforce output schema validation server-side.

## Summary

You understand where skills fit across CLI, web, and API surfaces.
