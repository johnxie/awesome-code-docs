# Chapter 1: Getting Started

This chapter gets Copilot CLI installed and launched for your first terminal agent session.

## Installation Paths

```bash
# macOS / Linux
brew install copilot-cli

# npm
npm install -g @github/copilot
```

Windows:

```powershell
winget install GitHub.Copilot
```

## First Launch

```bash
copilot
```

On first run, use `/login` if not already authenticated.
