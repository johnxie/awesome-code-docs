# Chapter 3: Task Pipelines

Master Turborepo's task pipeline system for efficient build orchestration and dependency management.

## Pipeline Basics

### Task Definition

```json
// turbo.json
{
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**"]
    },
    "test": {
      "dependsOn": ["build"]
    },
    "lint": {
      "dependsOn": ["^build"]
    }
  }
}
```

### Running Tasks

```bash
# Run specific task
turbo build

# Run with scope
turbo build --filter=apps/web

# Dry run
turbo build --dry-run
```

## Advanced Pipelines

### Complex Dependencies

```json
{
  "tasks": {
    "codegen": {
      "dependsOn": ["^codegen"]
    },
    "typecheck": {
      "dependsOn": ["codegen", "^build"]
    },
    "build": {
      "dependsOn": ["typecheck"]
    }
  }
}
```
