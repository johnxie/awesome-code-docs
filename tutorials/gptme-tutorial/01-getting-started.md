# Chapter 1: Getting Started

This chapter gets gptme installed and running in a local terminal session.

## Quick Install

```bash
pipx install gptme
# or
uv tool install gptme
```

## First Run

```bash
gptme
```

If provider keys are missing, gptme prompts to configure them.
