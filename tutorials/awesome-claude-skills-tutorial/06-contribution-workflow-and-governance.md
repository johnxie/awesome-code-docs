# Chapter 6: Contribution Workflow and Governance

This chapter explains how to contribute while preserving catalog quality.

## Learning Goals

- submit new skills with minimal maintainer friction
- avoid duplicates and category drift
- provide verifiable evidence of skill value
- align with repository contribution standards

## Contribution Flow

1. check for existing similar skills
2. follow structure/template requirements
3. place entry in proper category order
4. submit focused PR with clear context
