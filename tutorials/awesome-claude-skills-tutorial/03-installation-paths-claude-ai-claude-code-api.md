# Chapter 3: Installation Paths: Claude.ai, Claude Code, API

This chapter compares installation patterns across Claude usage surfaces.

## Learning Goals

- choose install path based on runtime and workflow
- verify skill activation in each environment
- avoid configuration drift across machines
- align local and team setup conventions

## Path Comparison

| Path | Typical Workflow |
|:-----|:-----------------|
| Claude.ai | marketplace/custom upload and activation |
| Claude Code | local skill folder placement and direct usage |
| API | skill ID references in programmatic calls |
