# Chapter 1: Getting Started

This chapter gets CCR installed and running with a minimal provider route.

## Learning Goals

- install the CLI and confirm command availability
- set up a valid base configuration
- route one request through CCR successfully
- verify restart behavior after config edits

## Baseline Commands

```bash
npm install -g @musistudio/claude-code-router
ccr start
ccr code
```
